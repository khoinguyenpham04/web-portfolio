## Overview

**ChessMind** is a sophisticated chess learning platform that combines interactive gameplay with AI-powered analysis and teaching capabilities. Built with Next.js and leveraging various modern technologies, this app offers a seamless user experience with advanced chess functionalities.

## Tech Stack

1. **Frontend Framework**:
   - Next.js 14.2 (App Router)
   - React 18
   - TypeScript

2. **Authentication**:
   - Clerk (`@clerk/nextjs`) for user authentication

3. **UI Components**:
   - Shadcn UI components
   - Radix UI primitives
   - Tailwind CSS for styling
   - Lucide React for icons
   - `class-variance-authority` for component variants
   - `tailwind-merge` for class merging

4. **Chess Related**:
   - `chess.js` for chess game logic
   - Stockfish.js for chess engine/AI
   - `react-chessboard` for chessboard UI

5. **AI/ML Integration**:
   - OpenAI API for move analysis
   - AI SDK for React (`@ai-sdk/react`)
   - Text-to-speech capabilities

## Key Features

1. **Chess Game**:
   - Interactive chessboard interface
   - Play against AI (Stockfish engine)
   - Multiple difficulty levels
   - Move validation
   - Undo/redo functionality
   - Game state management

2. **AI Analysis**:
   - Real-time move analysis using GPT-3.5
   - Text-to-speech explanations
   - Markdown-formatted analysis output
   - Strategic assessments and recommendations

3. **User Interface**:
   - Responsive design
   - Dark mode support
   - Animated components
   - Grid background with gradient effects
   - Custom button styles

4. **Authentication & Routes**:
   - Protected routes (/practice)
   - Sign in/Sign up pages
   - User session management

5. **Project Structure**:
   ```
   /app - Next.js app router pages
   /components - React components
   /hooks - Custom React hooks
   /lib - Utility functions
   /utils - Helper functions
   /public - Static assets
   ```

## Notable Components
